📐 Model Equations — Griffin et al. (2010) with demographic stochasticity

dS/dt  = μN + rT·Tr + rU·U − Λ·S − μ·S  // births + recoveries, minus new infections
dD/dt  = φ(ICA)·(1−fT)·Λ·(S+A+U) − (rD+μ)·D
dA/dt  = (1−φ)·Λ·(S+U) + rD·D − φ·Λ·A − (rA+μ)·A  // superinfection A→D
dU/dt  = rA·A − Λ·U − (rU+μ)·U  // all U bites exit U → D or A
b(IB)  = b₀·[b₁ + (1−b₁)/(1+(IB/IB₀)^kB)]  // Griffin 2010: saturating blood immunity
φ(ICA) = φ₀·[1 − ICA^kCA/(ICA^kCA + IC₅₀^kCA)]  // clinical immunity, ICA₅₀=8, never reaches zero
dIB/dt = EIR(t)/dB − IB/dB   // builds with bites (dB≈142d), mean-reverts
dICA/dt= incidence/(N·dCA) − ICA/dCA  // builds with clinical exposure (dCA≈662d)
SDE noise: du = f(u)dt + σ(u)·√dt·dW  // Euler-Maruyama; σ²=Σ|flux rates| (demographic stochasticity)
