# Project orientation

The purpose of this project is to examine the impact of different malaria control strategies realizing the cost and benefit of each of the approaches. This is part of a course for undergraduate students on parasitic disease control in the context of resource constraints (e.g., there are trade-offs that are a function of timing, budget, resources, and pathogen evolution given induced selective pressure from treatments and vaccination).

IMPORTANT: When citing literature, please refer explicitly to the source used as one of the sources.
